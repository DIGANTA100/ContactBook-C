# Contact Book in C

A simple contact management system built in C using file handling.

## Features

- Add new contact (Name, Phone, Email)
- Search contacts
- Show all contacts
- Menu-driven CLI

## How to Run

### Step 1: Compile 
```bash
gcc contact_book.c -o contact_book

```
### Step 2: Run
```
./contact_book
```
